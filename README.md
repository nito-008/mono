# Mono

Mono is a tiny toy web browser written in the [Rust](https://github.com/rust-lang/rust) language.

Goals:

- [ ] Render http://example.com

## How to run

### On WasabiOS (https://github.com/hikalium/wasabi)

```sh
./run_on_wasabi.sh
```

With noVNC:

```sh
make -C build/wasabi vnc # Default password is "wasabi".
```
